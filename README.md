# postgres-desktop-dbeaver-trigger

## Description
Creates a small database table
called `dog`.

## Tech stack
- postgres
- dbeaver

## Docker stack
- ubuntu:22.04
- postgres:alpine

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
